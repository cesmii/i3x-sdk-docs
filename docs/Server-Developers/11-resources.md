## Contributing

### How to Contribute

1. **Code Contributions**
   - Submit pull requests to https://github.com/cesmii/API
   - Follow the contribution guidelines
   - Add tests and documentation

2. **Documentation**
   - Improve existing documentation
   - Add examples
   - Translate documentation

3. **Bug Reports**
   - Use GitHub Issues
   - Provide detailed reproduction steps
   - Include version information

4. **Feature Requests**
   - Discuss in GitHub Discussions first
   - Submit detailed proposals
   - Consider backward compatibility

## Getting Help

### Before Asking for Help

1. Check the [FAQ](docs/faq.md)
2. Search [GitHub Issues](https://github.com/cesmii/API/issues)
3. Review the [documentation](https://github.com/cesmii/API)
4. Check [examples](docs/examples/)

### How to Ask for Help

1. Provide context and details
2. Include code samples
3. Describe what you've tried
4. Share error messages
5. Specify versions used

### Support Channels

- **Email**: rfc@cesmii.org (for RFC feedback)
- **Email**: devops@cesmii.org (for technical support)
- **GitHub Issues**: For bug reports
- **GitHub Discussions**: For questions and discussions
